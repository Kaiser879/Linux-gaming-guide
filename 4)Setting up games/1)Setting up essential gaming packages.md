Installing essential gaming packages on Cachy os is fairly simple
- Open Cachy os hello
- In Tweaks section you will find a tweak called install gaming packages 
- Run that tweak and it will install all essential gaming packages 
- If you choose not to run it and want to go with a minimal setup insted 
- ```bash
  sudo pacman -S steam gamemode lib32-gamemode mangohud lib32-mangohud wine winetricks lutris protonup-qt vulkan-tools heroic-games-launcher-bin //run this
  ```
- We will also need proton for games to run which we will install later 
- Steam Proton is a compatibility layer developed by Valve that allows Windows games to run on Linux-based operating systems through the Steam client. It combines a modified version of Wine with additional libraries to improve performance and compatibility for gaming.
